
Document (https://developers.google.com/docs/api/reference/rest/v1/documents)
Document contains body:
    - body contains "content": { object (StructuralElement) }
        - StructuralElement can be Table


Table (https://developers.google.com/docs/api/reference/rest/v1/documents#Table)
    - #rows
    - #cols
    - contains tableRows: [ {object (TableRow)}, ...]
        - TableRow: contains TableCells: [ {object (TableCell)}, ...]
            - TableCell: has "content": [ {object (StructuralElement)} ]
                - StructuralElement can be an object (Paragraph)
